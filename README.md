# RouteReady

RouteReady is a pre-drive confidence app. It shows you what your 
route looks like before you leave, and coaches you through every 
GPS dead zone. Built for drivers who need to know what's coming 
before it comes.

## Status
🚧 In Development — MVP Phase

## Stack
- React Native / Expo
- Google Maps Directions API
- Google Street View API
- OpenStreetMap (Overpass API)
- Supabase
- Claude API (Anthropic)
